## Built With

* Semantic HTML5
* CSS (Flexbox)
* Mobile-first workflow

## Design Choices

### Colors

* White: `#ffffff`
* Light Gray: `#d6e1f0`
* Grayish Blue: `#7f858a`
* Dark Blue: `#192644`

### Typography

* Font Family: *Outfit*
* Font Weights:

  * 400 (Regular)
  * 700 (Bold)

### Layout

* Centered card using Flexbox
* Fixed width container for consistency
* Rounded corners for modern UI look

## What I Learned

* How to structure semantic HTML properly
* How to center elements using Flexbox
* How to match a design closely using spacing, colors, and typography
